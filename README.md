# Mythology & Worldbuilding Scheduling and Booking Telegram Bot

Appointment booking, calendar sync, reminders, and waitlist management. No-app-needed booking for Mythology & Worldbuilding clients.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
